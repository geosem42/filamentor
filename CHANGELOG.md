# Changelog

All notable changes to `filamentor` will be documented in this file.

## [1.1.0] - 2025-05-30
### Added
- Enhanced page builder to allow dragging and dropping columns between different rows.
- Implemented new drag-and-drop handlers and Alpine.js store logic for cross-row column movement.
- Added CSS for visual drop indicators for columns between rows.

### Changed
- Updated Blade templates for the page builder UI to support new column drag-and-drop functionality.
- Refactored column reordering logic in the Alpine.js store.

## 1.0.0 - 2025-02-06

- Initial release
